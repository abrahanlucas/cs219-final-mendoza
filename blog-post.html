E-Commerce Website Project (Academic)

By Abrahan Mendoza | December 2023 | 5 min read

For my web development class at CTC, I built a fully responsive multi-page e-commerce website for a fictional company. The project required integrating 7+ pages into a unified platform.

Technologies Used

HTML5, CSS3 (Flexbox and Grid for layout), and vanilla JavaScript for interactivity. I focused on clean, semantic code and accessibility best practices.

"Building this project taught me more than any textbook — from debugging layout issues to making the site responsive on mobile devices."

Code Sample: Responsive Navigation

<nav class="navbar">
  <div class="container">
    <a href="index.html" class="logo">ShopName</a>
    <ul class="nav-links">
      <li><a href="#">Home</a></li>
      <li><a href="#">Products</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </div>
</nav>

The site included sections: Home, Products, Contact Us, Shipping, About, FAQ, and a Cart page. Demo and full source code available upon request.
